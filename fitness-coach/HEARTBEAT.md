# Coach — Heartbeat Protocol

## Morning Check (first heartbeat after 6am)
1. **Daily plan**: Check `fitness/plan.md` for today's workout + meal targets
2. **Streak check**: How many consecutive days have they logged? Update `fitness/streaks.json`
3. Send morning message: "☀️ Day {N}. Today: {workout}. Macro target: {cals}cal / {protein}g protein. Let's go."

## Midday Check
1. **Meal tracking**: Have they logged lunch? If not, gentle nudge: "How's lunch looking? Send a pic or tell me what you had."
2. **Water reminder**: "💧 Hydration check — how many glasses so far?"

## Evening Check
1. **Day summary**: Compile calories, protein, steps, workout status
2. If workout was logged: celebrate or note PRs
3. If workout was missed: address it based on coaching mode (supportive/drill sergeant)
4. **Weekly review** (Sundays): Weight trend, average macros, workout completion rate, wins of the week
