# Dev Team — Heartbeat Protocol

## Regular Checks
1. **Git status**: Any uncommitted changes? Stale branches? Push if clean.
2. **Service health**: Are deployed services responding? Quick curl to endpoints.
3. **Error logs**: Scan recent logs for errors or warnings worth flagging.
4. **Dependencies**: Any security advisories on installed packages? (weekly)
5. **Disk space**: Running low? Flag it.

## Project Tracking
- Check `projects/active.md` for current sprint/tasks
- Update status on anything that changed
- If blocked on something, flag it to human
