# TOOLS.md — Executive Command Center

## Daily Operations Workflow

### Morning Briefing Prep (7:30am)
1. **Calendar review** → Check conflicts, prep meeting materials
2. **Email triage** → Sort into urgent/action/FYI buckets
3. **Priority assessment** → Flag anything requiring immediate attention
4. **Brief prep** → 3-minute executive briefing ready
5. **Day optimization** → Suggest schedule adjustments if needed

## Email Management Templates

### Triage Categories
| Priority | Example | Response Time | Action |
|----------|---------|---------------|---------|
| 🔴 Urgent | Legal deadline, crisis | <1 hour | Forward with context |
| 🟡 Action | Meeting request, approval | <24 hours | Handle or escalate |
| 🟢 FYI | Industry news, updates | Weekly | Add to digest |
| 🗑️ Archive | Spam, irrelevant | Immediate | Handle without escalation |

### Response Templates


## Calendar Management Rules

### Scheduling Principles
1. **Focus blocks** → Minimum 2-hour deep work blocks daily
2. **Buffer time** → 15 minutes between meetings
3. **Travel time** → Account for location changes
4. **Prep time** → 30 minutes before important meetings
5. **Daily cutoff** → No meetings after [TIME] unless urgent

### Meeting Categories
- **Strategic** (CEO/Board level): Full prep packet, 30-min pre-brief
- **Operational** (Direct reports): Agenda required, action items tracked
- **External** (Partners/Clients): Background research, follow-up planned
- **Internal** (Team check-ins): Efficient, time-boxed, outcomes focused

## Travel Planning Checklist

### Pre-Booking Research
- [ ] Flight preferences (airline, seat, class)
- [ ] Hotel preferences (chain, room type, amenities)
- [ ] Ground transportation (rental, car service, airport transfer)
- [ ] Dietary restrictions and meeting meal requirements
- [ ] Time zone considerations and jet lag mitigation

### Booking Confirmation
- [ ] All confirmations saved with reference numbers
- [ ] Calendar updated with travel time blocks
- [ ] Mobile boarding passes and hotel apps installed
- [ ] Contact information for all vendors
- [ ] Backup options identified for critical connections

## Meeting Preparation Protocol

### 24 Hours Before
1. **Attendee confirmation** → Send calendar reminder with dial-in
2. **Materials package** → Agenda, background docs, attendee bios
3. **Logistics check** → Room booked, tech tested, catering ordered
4. **Executive brief** → 5-minute prep conversation scheduled

### Day Of
1. **Setup verification** → Room, tech, materials ready
2. **Attendee management** → Greet, direct, handle latecomers
3. **Note taking** → Action items, decisions, next steps
4. **Follow-up prep** → Action items ready for distribution

### Within 2 Hours After
1. **Action items distributed** → Clear owners and deadlines
2. **Next steps scheduled** → Follow-up meetings in calendar
3. **Executive debrief** → Key outcomes and concerns noted
