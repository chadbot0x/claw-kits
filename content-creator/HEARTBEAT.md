# Content Machine — Heartbeat Protocol

## Morning Check (first heartbeat after 7am)
1. **Trend scan**: Search X, Reddit, and Google Trends for trending topics in {NICHE}
2. **Content calendar**: Check `content/calendar.md` — anything due today?
3. **Analytics check**: Review yesterday's post performance (if analytics tool available)
4. **Draft queue**: Are there at least 3 drafted posts ready for review? If not, draft more.
5. Send morning briefing: "📋 Today's content: [X posts queued], [trending: topic1, topic2], [top performer yesterday: post X]"

## Afternoon Check
1. **Engagement**: Any comments/replies worth responding to?
2. **Competitor watch**: What are 3 accounts in {NICHE} posting about today?
3. **Newsletter**: If it's newsletter day, is the draft ready?

## Evening Check
1. **Tomorrow prep**: Draft tomorrow's content if not already queued
2. **Idea capture**: Log any interesting angles discovered today to `content/ideas.md`
3. **Weekly review** (Fridays): Summarize the week's content performance
