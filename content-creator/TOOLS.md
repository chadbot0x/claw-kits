# TOOLS.md — Content Machine

## Content Pipeline

### Workflow
1. **Research** → web_search for trends, fetch competitor posts
2. **Draft** → Write to `content/drafts/YYYY-MM-DD-{slug}.md`
3. **Review** → Send draft summary to human via message
4. **Publish** → After approval, post via platform tools/browser
5. **Track** → Log published posts in `content/published.md`

### File Structure
```
content/
  calendar.md        # Weekly content calendar
  ideas.md           # Running idea backlog
  voice-guide.md     # Brand voice reference + examples
  drafts/            # Ready-to-review drafts
  published.md       # Log of what went live + performance
  templates/         # Reusable post formats
```

## Platform Formatting
- **X/Twitter**: Max impact in 280 chars. Hooks matter. No hashtag spam (1-2 max). Threads: each tweet must stand alone.
- **LinkedIn**: Professional but human. Stories perform. No emoji walls. First 3 lines are the hook (before "see more").
- **Newsletter**: Subject line is everything. One main topic per issue. Personal anecdotes convert.
- **Instagram**: Caption hooks in first line. 3-5 relevant hashtags at the end. Carousel > single image.

## Repurposing Matrix
| Source | → X | → LinkedIn | → Newsletter | → Short Video |
|--------|-----|-----------|-------------|---------------|
| Blog post | Pull 3-5 key quotes | Rewrite intro as story | Summarize + link | Key points as talking head |
| Podcast | Timestamp best moments | Guest quote + lesson | Transcript highlights | Clip best 60s |
| Thread | Already done | Expand into article | Compile + add depth | Narrate over B-roll |

## Hooks That Work
- "I {did X}. Here's what happened."
- "Stop doing {common mistake}."
- "The {industry} doesn't want you to know this."
- "{Number} things I learned after {achievement}."
- "Unpopular opinion: {take}"
- "Everyone is talking about {trend}. Here's what they're missing."
