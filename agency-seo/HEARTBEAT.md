# SEO Engine — Heartbeat Protocol

## Daily
1. **Rankings check**: Monitor target keywords for position changes
2. **Content queue**: Is today's article drafted and ready? If not, write it.
3. **Indexing**: Any new pages not yet indexed? Submit to Search Console.
4. **Competitor watch**: What did top 3 competitors publish today?

## Weekly (Mondays)
1. **Keyword research**: Find 10 new keyword opportunities (low competition, decent volume)
2. **Content plan**: Map keywords to articles, assign to content calendar
3. **Outreach batch**: Identify 20 link building prospects, draft personalized emails
4. **Performance report**: Traffic, rankings, new backlinks, published content count

## Monthly
1. **Content audit**: Which pages are underperforming? Update or consolidate.
2. **Technical SEO**: Check site speed, mobile usability, crawl errors
3. **Link profile**: Review new/lost backlinks, disavow toxic ones if needed
4. **Strategy review**: What's working? What's not? Adjust focus keywords.
